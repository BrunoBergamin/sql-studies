# Glossario basico de SQL

## O que e SQL

SQL significa `Structured Query Language`.
E a linguagem usada para criar, consultar e organizar dados em bancos relacionais.

## Comandos iniciais

`CREATE DATABASE`
Cria um banco de dados.

`USE`
Seleciona o banco que sera usado.

`CREATE TABLE`
Cria uma tabela.

`INSERT INTO`
Insere novos registros.

`SELECT`
Consulta dados.

`WHERE`
Filtra resultados.

`ORDER BY`
Ordena os resultados.

## Palavras importantes

`PRIMARY KEY`
Identifica de forma unica cada registro.

`VARCHAR`
Tipo de dado para texto.

`INT`
Tipo de dado para numeros inteiros.

`DECIMAL`
Tipo de dado para numeros com casas decimais.
